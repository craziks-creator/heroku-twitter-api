Welcome to Hello Twitter!

A simple web application that allows a user to interact with Twitter using their developer API

Let's Go!

Overview:
  1. This will show a basic demo of the standard Twitter 3-legged application authorization flow (also known as Sign in with Twitter).
  2. This is a basic demo created by the @TwitterDev team. It is written in Python, using Flask and python-oauth2. Basic CSS is provided by Bootstrap. The source code for this project can be found at github/twitterdev/twauth-web.
Requirements:
  1. A valid Twitter account
  2. Curiosity
Here's our plan:
  1. Generate an OAuth request token to be granted access.
  2. Rdirect you to Twitter.com using our tokens from step 1. You'll then be prompted to authorize the app.
  3. Once authorized, you'll then be redirected back to the Hello Twitter application.
